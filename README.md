# 42 Piscine Python

🎓 **42 Piscine Python** — A collection of Python exercises and projects completed during the 42 School Python Piscine.

This repository contains the different modules of the Python Piscine, organized by folders (00, 01, 02, …). The goal is to progressively learn Python, from basic syntax to more advanced concepts such as data structures and object-oriented programming.

---

## 🧠 Objectives

The Python Piscine aims to:

* Learn Python fundamentals
* Practice programming through hands-on exercises
* Understand core concepts such as functions, classes, modules, exceptions, and file handling
* Build a strong foundation for future 42 projects

## 🛠️ Installation & Usage

### Requirements

* Python ^3.10
* A terminal (bash, zsh, PowerShell, etc.)
* (Optional) A virtual environment

### Setup (if needed)

If a `requirements.txt` file is provided:

python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

### Run an Exercise

Navigate to a specific module:

cd 00
python3 main.py

(Replace `main.py` with the correct file depending on the exercise.)

✔️ Read the subject carefully before coding
